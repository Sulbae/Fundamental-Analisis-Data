# Fundamental-Analisis-Data

## Setup Environment - Anaconda
1) `conda create --name main-ds python=3.12`
2) `conda activate main-ds`
3) `pip install -r requirements.txt`

## Run Streamlit
`streamlit run dasboard.py`
